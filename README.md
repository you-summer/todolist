# 🐣 Egg To-Do List

> **"당신의 알을 깨워보세요!"**  
> 작은 알처럼 소중한 할 일을 하나하나 부화시키는 감성 투두리스트 앱입니다.
---

### 🛠️ 사용 기술

- React (create-react-app 기반)
- JavaScript (ES6+)
- CSS
- Context API 
- localStorage 
- Vercel

---

## ✨ 주요 기능

- ✅ 할 일 추가 및 삭제
- ✅ 검색 기능
- ✅ 오늘 날짜 표시
- ✅ 실시간 시계 표시
- ✅ 완료 체크 (줄긋기 표시)
- ✅ 로컬스토리지 저장
- ✅ 다크모드 (아이콘: 🌙 / ☀️)
- ✅ 통계/개수 표시
- ✅ 배포
